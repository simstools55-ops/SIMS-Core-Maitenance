# SIMS_REQUEST_TEXT_V1

## 1. 目的

SIMS-Blog-ManagerからClaudeまたは他の生成AIへ渡す、可読性と移植性を優先した記事改善依頼形式です。

## 2. 必須項目

依頼文には原則として次を含めます。

- 改善要求文
- ArticleID
- URL
- 記事タイトル
- SEOタイトル
- メタディスクリプション
- メインクエリ
- 現在値：クリック、表示、CTR、平均順位
- 改善目的
- SIMS_FEEDBACK_V1の出力ルール

## 3. 任意入力

必要に応じて次を追加できます。

- 記事本文
- 見出し一覧
- 関連クエリ
- FAQ候補
- 内部リンク候補
- CMS・テーマ固有条件
- 変更禁止箇所

## 4. 解釈規則

1. ArticleIDとURLは識別子として原文のまま保持する。
2. 数値を推測・補完しない。
3. 改善目的を最優先し、明示されていない全面リライトを勝手に行わない。
4. 「既存記事の良い部分は残す」とある場合、差し替え対象と維持対象を区別する。
5. 記事本文が未提供の場合、URLを参照できる環境では本文を確認し、参照できない場合は不足を明示して可能な範囲を改善する。
6. 出力JSONの仕様は依頼文中の指定を優先する。

## 5. 互換性

本形式は自然文であるため、Claude固有の機能を必須としません。対応AIは、依頼本文と末尾のJSON契約を理解できれば実装可能です。
