# Knowledge Architecture

Claude Projectへ登録するKnowledgeは、旧版の多数のEngine・テンプレートをそのまま投入せず、役割別に統合します。

1. `01_SIMS_CORE_RULES.md`：役割、入力解釈、処理順、禁止事項
2. `02_SEO_IMPROVEMENT_GUIDE.md`：検索意図、メタデータ、構成、FAQ、本文改善
3. `03_ARTICLE_WRITING_GUIDE.md`：文章品質、既存文維持、コピー可能な出力
4. `04_QUALITY_ASSURANCE.md`：最終検査と整合性
5. `05_OUTPUT_SPECIFICATION.md`：通常出力とFeedback JSON契約
6. `06_BLOG_PROFILE.md`：ブログ固有ルールを記載する利用者編集領域

KnowledgeはAIの判断補助です。Blog-Manager、Apps Script、GitHub Pagesの実装詳細は含めません。
