# Claude Projectセットアップ

1. ClaudeでSIMS-Core用Projectを作成します。
2. `distribution/SIMS-Core-Claude-Upload/01_Project_Instructions/CLAUDE_PROJECT_INSTRUCTIONS.md` を開き、本文をProjectの「手順」へ貼り付けます。
3. `02_Project_Knowledge` のMarkdownファイルをProjectの「コンテキスト」へ追加します。
4. `06_BLOG_PROFILE.md` はブログ固有ルールがある場合だけ編集して差し替えます。
5. SIMS-Blog-Managerが生成した改善依頼をチャットへ貼り付けます。
6. 回答末尾のJSONをBlog-Managerの結果登録に使用します。

旧SIMS-Claude-EngineのEngine01〜10や大量のテンプレートを同時に登録しないでください。旧ルールとの競合を避けるため、Production配布フォルダのファイルだけを使用します。
